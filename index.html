⚔ Fantasy Card Battle ⚔

昔ながらのファンタジー世界をテーマにした MtG ライクカードゲーム

🐍 Python 3  |  プレイヤー vs CPU  |  ランダムデッキ

🚀 インストール & 起動

# リポジトリをクローン
git clone <this-repo>
cd Claude-card

# 依存パッケージのインストール
pip install -r requirements.txt

# ゲーム起動
python main.py

📜 ゲームルール

  • デッキ — 各プレイヤーは30枚のランダムデッキ（土地12・クリーチャー10・呪文8）を使用
  • 初期ライフ — 20点。先手は最初のドローなし
  • マナ — 土地をタップして1マナを生み出す。1ターンに土地1枚まで設置可能
  • ターン進行 — アンタップ → ドロー → メインフェーズ（カードプレイ） → 戦闘 → 終了
  • 勝利条件 — 相手のライフを 0 以下にする

✨ 特殊能力

◎警戒攻撃してもタップしない
